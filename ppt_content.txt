Problem Statement
Android devices are widely used, making them a major target for malware attacks.
Many users unknowingly install malicious applications that steal personal data, track activities, or damage the system.
Existing detection methods require technical knowledge or expensive security tools .
Because of this, it becomes difficult for normal users to identify malware threats in APK files, 
wifi
 networks, or suspicious messages.
Therefore, a smart and automated system is required to detect malware quickly and accurately.

Abstract
The proposed project, 
ShieldX
 – Android Malware Detection System, aims to develop a smart web-based platform that detects malware using Machine Learning techniques. The system allows users to upload malware datasets, train machine learning models, and perform threat analysis through an interactive dashboard.
The proposed solution uses machine learning algorithms such as Random Forest and Logistic Regression to analyze patterns in malware datasets and predict whether an application is malicious or safe. The platform also provides additional features such as scanning APK files, 
WiFi
 networks, and suspicious messages to detect potential threats.
